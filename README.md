# 📊 Repositorio de Prácticas y Tareas — TripleTen LATAM

## 👋 Introducción
Este repositorio contiene una recopilación de tareas, ejercicios y análisis desarrollados en formato **Jupyter Notebook** como parte de mi proceso de formación en análisis de datos con **TripleTen LATAM**.

El objetivo principal es documentar mi progreso, reforzar habilidades técnicas y construir un portafolio sólido basado en proyectos prácticos y casos reales.

---

## 📚 Sobre el contenido
Aquí encontrarás notebooks que incluyen:

- 📌 Análisis exploratorio de datos (EDA)  
- 📊 Limpieza y transformación de datos  
- 📈 Visualización de datos  
- 🧠 Aplicación de conceptos estadísticos  
- 🛠️ Uso de herramientas como Python, Pandas y NumPy  

Cada notebook está estructurado para explicar paso a paso el proceso analítico, incluyendo contexto, desarrollo y conclusiones.
